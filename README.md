# 🧠 Logic Box – Pattern Generator & Number Analyzer

## 📌 Overview

Logic Box is a simple Python console application that demonstrates the use of loops, conditional statements, and user input handling. The program provides two useful functionalities:

1. **Pattern Generator**
2. **Number Range Analyzer**

This project is beginner-friendly and helps in understanding core Python programming concepts.

---

## ✨ Features

### 🔹 Pattern Generator

* Generates a star (`*`) pattern based on the number of rows entered by the user.
* Uses nested loops for pattern creation.

Example:

```text
*
* *
* * *
* * * *
```

### 🔹 Number Analyzer

* Analyzes numbers within a user-defined range.
* Identifies whether each number is **even** or **odd**.
* Calculates the **sum of all numbers** in the specified range.

Example:

```text
1 is odd
2 is even
3 is odd
4 is even

Sum of all numbers from 1 to 4 : 10
```

### 🔹 Exit Option

* Allows users to safely terminate the program.

## ▶️ How to Run

1. Open the notebook in Jupyter Notebook or JupyterLab.
2. Run all cells.
3. Choose an option from the menu:

   * `1` → Generate Pattern
   * `2` → Analyze Range of Numbers
   * `3` → Exit Program
